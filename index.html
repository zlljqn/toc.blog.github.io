HEXO+GITHUB+NexT搭建个人Blog （∂Mac）

发表于 2017-01-27 | 分类于  技术学习 |

前面随笔说了为什么我要搭建一个个人的Blog，这里也就不赘述了，反正撩起袖子就干活，想干的事一刻都等不了，整个搭建大概用了5个小时，大坑没有，小坑还是有一些。

注意：本文使用Hexo搭建Blog，Hexo是基于Node.js的静态博客，通俗的讲，静态的意思是需要在本地编辑文章的Mackdown页面，每次通过Node.js编译生成整个站点的静态内容，再发布到线上，是没有办法在线上直接编辑撰写博客的。所以普通用户读到这里，如果觉得对自己Markdown水品和技术水平不那么自信的话，读到这里可以离场了。

环境介绍

操作系统：
* MacOS v10.12.2

网络VPN：
* GreenVPN - 很多站点功能都在国外，需要VPN翻墙，GreenVPN是在MAC上我用的顺手的。

软件：
* GIT v2.10.1
* Node.js v6.9.4
* Hexo v3.3.2

Hexo主题：
* NexT

资源介绍

* GitHub - 用于存放Blog站点文件和文章（https://github.com/）
* GoDaddy - 购买自己心仪的域名，管理域名DNS（https://sg.godaddy.com/zh/）
*极简图床 - 用于存放Blog中需要的图片（http://yotuku.cn/）

搭建前准备

  1. VPN翻墙
    虽然很多被墙的开源站点获软件可以用国内镜像替代，但是还是建议使用VPN翻墙，我用顺手的是GreenVPN；GreenVPN在百度上一搜，会出来好多站点，很多是山寨的，我用的是https://www.getgreenjsq.co/index.php，如下图：

    接下来安网站指引注册，购买，安装（这里不细讲了），完成后，会在系统的右上角显示一片叶子，点击叶子可以登录，打开和关闭VPN。

    好了，你现在可以去试试能不能访问Google了。
  2. 购买域名（可选）
    在GitHub上的Pages部署Blog的话，GitHub会给Pages一个默认的访问地址，但这个地址正常人一般会感觉非常拗口难记，类似于https://[username].github.io/[pagename].github.io/。另外自己的Blog应该要有自己的名字，我们可以购买自己心仪的域名来访问我们的Blog，比如我使用了[http://blog.tocstudio.net][5]作为我Blog的访问地址。
    域名购买推荐使用GoDaddy，活动多，价格不贵，管理方便。GoDaddy中文站点地址：https://sg.godaddy.com/zh。通过搜索域名查找自己想要的域名是否还可以使用购买。后文步骤中再讲解域名如何使用。

新的开始，不变的梦想

发表于 2017-01-26 | 分类于  随笔 |

倒计时170天

今天勉强算是一个纪念日吧，看看今天到6月底也是一个整数倒计时。

不变的梦想

目标和梦想，想了一下也就认真过2次；一次小学时期，励志当老师，一直认真到考大学，可惜梦想有，没有真努力，没考上师范；另一次，刚进大学那时，失去了原来的目标又碰到很多糟心事，处于最低谷，想了好几天人生是什么，然后确定了自己下一个人生目标和梦想：一定要在历史长河中留下我的足迹。这个梦想支撑了现在不断的挑战、探索、尝试，至今没有变。

关于改变

人生中我能记得的变化有几次，一个手也够数完，有好有坏，坏的多点（毕竟挫折更能改变一个人），但想想不算太坎坷，但也可以说不精彩；幸运每次变化都挺住，没有飘到很高，也没长时间颓废，还算在变化中得到不少成长，借用一句话：What doesn’t kill me make me stronger. （ 大声说一句：ßßß I couldn’t agree more！！！）

新的开始

我相信现在我也处于一个新的低谷，但是我也嗅到了这样一个低谷为我挤压出的机遇，“是时候改变了！”；能让我在40岁前还能惊醒，我还是非常感谢这个及时到来的低谷，之前舒服的生活走的太长了，思想也就疲惫了；索性让我冷静下来找到初心，拾起那一直在却好久没有触碰的梦想。这一刻，新的旅程已经开始，虽然前方是另一片迷朦，但好在这次出发有好兄弟，有好思路，有家人支持，可以向自己的梦想再挪一小步，想想就算这次路再坎坷，也应该是幸福的。

小记

Blog那么想想应该是已经过时的流行了，现在弄起来，其实也是为之后的开始做个热身。域名、服务器、程序、部署、设计、文稿撰写，过程在技术分享里面记录吧。
